# 🏭 Интерактивная система управления производством

Веб-приложение для автоматизации управления производственными процессами, разработанное на **ASP.NET Core MVC**.

## 🚀 Быстрый старт

### Запуск приложения
```bash
cd C:\Users\nolik\Desktop\PKS\ProductionManagement
dotnet run
```

### Доступ к приложению
- **URL:** http://localhost:5000
- **API:** http://localhost:5000/api/

## 📋 Основные возможности

### 1. 📊 Дашборд
- Статистика по активным заказам
- Количество работающих линий
- Материалы с низким запасом
- Последние заказы

### 2. 🧱 Управление материалами
- Просмотр всех материалов со статусом запасов
- Добавление новых материалов
- Редактирование параметров
- Пополнение запасов
- Цветовая индикация низких запасов

### 3. 📦 Каталог продуктов
- Просмотр всех продуктов
- Фильтр по категориям
- Поиск по названию
- Управление материалами для каждого продукта
- Установка времени производства

### 4. 📋 Управление заказами
- Создание новых производственных заказов
- Автоматический расчет времени выполнения
- Отслеживание прогресса
- Фильтрация по статусам
- Кнопки запуска и отмены заказов

### 5. ⚙️ Управление производственными линиями
- Визуальное отображение всех линий
- Управление статусом (Активна/Остановлена)
- Регулировка коэффициента эффективности
- Расписание предстоящих заказов
- Отслеживание текущего заказа

## 🔌 API Endpoints

### Материалы
```
GET    /api/materials                    - Получить все материалы
GET    /api/materials?low_stock=true     - Только с низким запасом
POST   /api/materials                    - Добавить материал
PUT    /api/materials/{id}/stock         - Изменить количество
```

### Продукты
```
GET    /api/products                     - Получить все продукты
GET    /api/products?category=...        - Фильтр по категории
GET    /api/products/{id}/materials      - Материалы продукта
POST   /api/products                     - Создать продукт
```

### Производственные линии
```
GET    /api/lines                        - Все линии
GET    /api/lines?available=true         - Только доступные
GET    /api/lines/{id}/schedule          - Расписание линии
PUT    /api/lines/{id}/status            - Изменить статус
PUT    /api/lines/{id}/efficiency        - Изменить коэффициент
```

### Производственные заказы
```
GET    /api/orders                       - Все заказы
GET    /api/orders?status=active         - Активные заказы
GET    /api/orders?status=active&date=today - Заказы на сегодня
GET    /api/orders/{id}/details          - Полная информация
POST   /api/orders                       - Создать заказ
PUT    /api/orders/{id}/progress         - Обновить прогресс
POST   /api/orders/{id}/start            - Запустить заказ
POST   /api/orders/{id}/cancel           - Отменить заказ
```

### Расчеты
```
POST   /api/calculate/production         - Рассчитать время
```

## ⚡ Ключевые формулы

### Расчет времени производства
```
Время (минут) = (Количество × ВремяНаЕдиницу) / КоэффициентЭффективности
```

Пример: 5 единиц × 120 мин/шт ÷ 1.2 = 500 минут = 8.33 часа

### Проверка материалов
```
Необходимо = QuantityNeeded × КоличествоЗаказа
Доступно = Material.Quantity
Проверка = Необходимо ≤ Доступно
```

## 🎨 Цветовые коды

- 🟢 **Green** - Active/Completed/Норма
- 🔵 **Blue** - InProgress/Pending
- 🔴 **Red** - Cancelled/Low Stock
- 🟡 **Yellow** - Overdue/Warning

## 📝 Примеры использования

### Создание нового заказа
1. Перейти в раздел "Заказы"
2. Нажать кнопку "+ Создать заказ"
3. Выбрать продукт (время автоматически заполнится)
4. Ввести количество (сроки пересчитаются)
5. Выбрать производственную линию (опционально)
6. Нажать "Создать заказ"

### Добавление материала к продукту
1. Перейти в "Продукты"
2. Нажать на продукт (кнопка "Открыть")
3. Нажать "+ Добавить" в разделе "Необходимые материалы"
4. Выбрать материал и количество
5. Нажать "Добавить"

### Управление линией
1. Перейти в "Линии"
2. Использовать ползунок для изменения коэффициента эффективности
3. Нажать "Запустить линию" или "Остановить линию"
4. Просмотреть расписание предстоящих заказов

## 📲 API примеры

### Получить активные заказы
```bash
curl http://localhost:5000/api/orders?status=active
```

### Создать заказ
```bash
curl -X POST http://localhost:5000/api/orders \
  -H "Content-Type: application/json" \
  -d '{
    "productId": 1,
    "quantity": 5,
    "lineId": 1
  }'
```

### Рассчитать время производства
```bash
curl -X POST http://localhost:5000/api/calculate/production \
  -H "Content-Type: application/json" \
  -d '{
    "productId": 1,
    "quantity": 10
  }'
```
