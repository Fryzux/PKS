# Проверка требований - Интерактивная система управления производством

## 1. ТРЕБОВАНИЯ К БАЗЕ ДАННЫХ (Entity Framework Core)

### 1.1 Product (Продукт)
- [x] Id (int) - уникальный идентификатор продукта
- [x] Name (string) - наименование продукта (обязательное поле)
- [x] Description (string) - текстовое описание продукта
- [x] Specifications (string/JSON) - технические характеристики в формате JSON
- [x] Category (string) - категория продукта
- [x] MinimalStock (int) - минимальный необходимый запас на складе
- [x] ProductionTimePerUnit (int) - время производства одной единицы (в минутах)

### 1.2 ProductionLine (Производственная линия)
- [x] Id (int) - уникальный идентификатор линии
- [x] Name (string) - название производственной линии
- [x] Status (string) - текущий статус ("Active"/"Stopped")
- [x] EfficiencyFactor (float) - коэффициент эффективности (0.5-2.0)
- [x] CurrentWorkOrderId (int?) - ID текущего выполняемого заказа (может быть null)

### 1.3 Material (Материал)
- [x] Id (int) - уникальный идентификатор материала
- [x] Name (string) - наименование материала
- [x] Quantity (decimal) - текущее количество на складе
- [x] UnitOfMeasure (string) - единица измерения ("кг", "шт", "литр" и т.д.)
- [x] MinimalStock (decimal) - минимальный запас для пополнения

### 1.4 ProductMaterial (Связь продукта и материала)
- [x] ProductId (int) - ссылка на продукт (внешний ключ)
- [x] MaterialId (int) - ссылка на материал (внешний ключ)
- [x] QuantityNeeded (decimal) - необходимое количество материала для производства одной единицы продукта

### 1.5 WorkOrder (Производственный заказ)
- [x] Id (int) - уникальный идентификатор заказа
- [x] ProductId (int) - ссылка на производимый продукт
- [x] ProductionLineId (int?) - назначенная производственная линия (может быть null)
- [x] Quantity (int) - количество единиц к производству
- [x] StartDate (DateTime) - плановая дата начала производства
- [x] EstimatedEndDate (DateTime) - расчетная дата завершения
- [x] Status (string) - статус выполнения ("Pending"/"InProgress"/"Completed"/"Cancelled")
- [x] Progress (int) - прогресс выполнения (добавлено для отслеживания)

## 2. API СПЕЦИФИКАЦИЯ

### 2.1 Материалы
- [x] GET /api/materials?low_stock=true - Получить список материалов (только с низким запасом)
- [x] POST /api/materials {name, quantity, unit, min_stock} - Добавить новый материал
- [x] PUT /api/materials/{id}/stock {amount} - Изменить количество на складе

### 2.2 Продукты
- [x] GET /api/products?category={id} - Список продуктов (с фильтром по категории)
- [x] GET /api/products/{id}/materials - Получить материалы для продукта
- [x] POST /api/products {name, prod_time, category} - Создать новый продукт

### 2.3 Производственные линии
- [x] GET /api/lines?available=true - Список линий (только доступные)
- [x] PUT /api/lines/{id}/status {status} - Изменить статус линии
- [x] GET /api/lines/{id}/schedule - Получить расписание линии
- [x] PUT /api/lines/{id}/efficiency {factor} - Изменить коэффициент эффективности

### 2.4 Заказы
- [x] GET /api/orders?status=active&date=today - Фильтрация заказов
- [x] POST /api/orders {product_id, quantity, line_id} - Создать заказ с авторасчетом
- [x] PUT /api/orders/{id}/progress {percent} - Обновить прогресс выполнения
- [x] GET /api/orders/{id}/details - Полная информация по заказу
- [x] POST /api/orders/{id}/start - Запустить заказ в производство
- [x] POST /api/orders/{id}/cancel - Отменить заказ

### 2.5 Расчетные методы
- [x] POST /api/calculate/production {product_id, quantity} - Рассчитать время производства

### 4.1 Расчет времени выполнения
- [x] Формула: Время = (Количество × ВремяНаЕдиницу) / КоэффЭффективности
- [x] Реализовано в сервисе ProductionCalculationService
- [x] Используется в API и MVC контроллерах

### 4.2 Контроль материалов
- [x] Автоматическая проверка при создании заказа
- [x] Реализовано в сервисе MaterialValidationService
- [x] Возвращает список ошибок при недостаточности материалов
